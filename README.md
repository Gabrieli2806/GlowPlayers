# Glowplayers

a plugin designed to enhance player interaction and visibility by automatically assigning a glowing effect to players upon joining the server. Each player is given a unique color based on the order in which they connect, making it easy to identify individuals in a group or event.

## How to Use

See the [releases page](https://github.com/Gabrieli2806/GlowPlayers/releases) to download and get started.
