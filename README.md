# GlowPlayers

GlowPlayers is a plugin designed to enhance player interaction and visibility by automatically assigning a glowing effect to players upon joining the server. Each player is given a unique color based on the order in which they connect, making it easy to identify individuals in a group or event.

## How to Use
1. Download the GlowPlayers plugin and place it in your server's plugins directory.
2. Restart your server to enable the plugin.
3. Players will automatically receive a glowing effect with unique colors upon joining.
